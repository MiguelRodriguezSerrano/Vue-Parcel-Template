# Plantilla Vue - Vue-Router - Vuex - Parcel

Plantilla para el desarrollo con Vue (incluyendo configuracion de Vuex y Vue-Router), usando ParcelJS como Bundler.

## Instalación:

`cd Vue-Parcel-Template`
<br />
`npm install`

## Desarrollo:

`npm run dev`

## Producción:

`npm run build`
